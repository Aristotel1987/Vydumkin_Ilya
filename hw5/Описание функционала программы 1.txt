Модель Отеля.
Программа состоит из объектов 6 классов.
1. базовый класс для комнат. Включаед следуюхщие атрибуты:  нмер комнаты, тип комнаты, максимальное число гостей, цена за ночь, доступность(по умолчанию комната доступна). У базового классса есть следующие методы: забронировать комнату и освободить комнату.
2. Класс люкс. Наследует базовому классу. Добавлены атрибуты наличия балкона и мини-бара.  Переопределен метод бронирования, изменено информационное сообщение при бронировании.
3. Класс стандарт. Наследует базовому классу.  ВВеден атрибут количества занятых мест. Переопределены методы бронирования и освобождения комнаты. Ведется учет свободных и занятых мест. Номер считается свободным, пока  есть хотя бы одно свободное место.
4. Класс Эконом, наследует Стандартному, ничем не отличается, создан для иллюстрации последовательного наследования.
5. Класс Гость.  Имеет следущие атрибуты: полное имя, телефон, номер брони, номер комнаты.  Есть одни метод, бронирование указанной комнаты.
6. Класс Отель Имеет единственный атрибут - список номеров и следующие методы: выводвсего списка номеров, поиск свободных номеров, поиск номеров для заданного числа гостей. 
Далее в программе объекту hotel присваивается класс Отель и происходит заполнение его номерами, при помощи ввода с клавиатуры. После чего происходит ввод данных о госте,и заселение гостя в указанный номер,  поиск  номера для введенного числа гостей, с результатом поиска, выведенным на экран.
